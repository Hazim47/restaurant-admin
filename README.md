🍔 Shawarma Admin Panel – Restaurant Management System
📱 Overview

Shawarma Admin Panel is a powerful and simple restaurant management system designed to control and manage all incoming orders from the customer application in real-time. It allows restaurant owners and staff to efficiently handle orders, update statuses, and manage menu items from a single dashboard.

This project is part of a complete restaurant ordering system.

1- Features
   Real-time order management
   Update order status (new / preparing / ready / delivered)
   View detailed order information
   Manage menu items (add / edit / delete)
   Simple and clean admin dashboard
   Fast and responsive UI
   Connected with customer ordering app
2-Tech Stack
    Backend integration (Supabase / Firebase / API )
    Real-time database synchronization
    Flutter
     
3- Portfolio Project

    Mobile app development with Flutter
    Real-time database integration
    UI/UX design for admin systems
    Full system architecture (Customer + Admin apps)
    Backend integration (Supabase / Firebase / API)
    Real-time database synchronization
    
4- Purpose
This admin panel is designed to give restaurant owners full control over their business operations without relying on third-party platforms. It improves efficiency, reduces order mistakes, and provides a centralized system for managing everything.
